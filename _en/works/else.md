## Religious Painting #5 (Prototype)

Video game
{: .label }
Tangible interaction
{: .label .label-green }
Physical computing
{: .label .label-green }
Interactive CG
{: .label .label-green }
Brightness sensor
{: .label .label-green }
Touch sensor
{: .label .label-green }
Unity
{: .label .label-purple }
Arduino
{: .label .label-purple }
Blender
{: .label .label-purple }
v2020-07-09
{: .label .label-yellow }

- A video game where you solve "escape room" puzzles in a digital world through analog controls.
  - Lowering the value of a brightness sensor darkens the lighting in the game room, revealing hints.
  - Turning a potentiometer knob rotates the land in the game, allowing movement to previously inaccessible areas.
  - Grabbing a touch sensor makes the player grab a pole in the game to avoid falling. Puzzles are solved through these kinds of interactions.
- Developed using Unity, Arduino, and Blender.
- ([Source Code](https://github.com/random-entity/o.art.religious-painting-5))

## Interactive Logo Design: Hongkong Banjum 0410

Interactive design
{: .label }
Physics simulation
{: .label .label-green }
Interactive CG
{: .label .label-green }
Processing
{: .label .label-purple }
v2019
{: .label .label-yellow }

- An interactive logo design for "Hongkong Banjum 0410". Holding the space bar turns the mouse cursor into a banknote. Cutting a bundle of noodles with the banknote makes them fall into the bowl below, transforming into wriggling mussel jjamppong (spicy seafood noodle soup).
- Developed using Processing and the [toxiclibs] library.
- [(Source Code)](https://github.com/random-entity/o.des.hongkong)

## Sound Interaction Design: Playground and Whistle

Video game
{: .label }
Interactive CG
{: .label .label-green }
Sound interaction
{: .label .label-green }
Automata
{: .label .label-green }
Processing
{: .label .label-purple }
v2019-11-12
{: .label .label-yellow }

- Children are scattered around the playground, playing in groups. When a (real-world) whistle blows, the children align into neat rows and columns. If you blow the whistle at regular intervals, the children perform gymnastics to that rhythm. When the gymnastics end, the children scatter and play again.
- Graphic Design: Haein Kim; Programming: Random Entity
- Developed using Processing, [toxiclibs], and the Sound library.
- [(Source Code)](https://github.com/random-entity/o.des.horuragi)
