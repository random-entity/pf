- 아날로그 조작으로 디지털 세계 속 "방탈출" 퍼즐을 풀어 나가는 비디오게임. -조도 센서의 값을 낮추면 게임 속 방의 조명이 어두워져 힌트가 보이게 된다든가,
  - 가변저항 노브를 돌리면 게임 속 대지가 회전해 이동할 수 없었던 곳으로 이동할 수 있게 된다든가,
  - 터치 센서를 잡으면 게임 속에서 플레이어가 봉을 잡아 떨어지지 않게 된다든가 하는 조작을 통해 퍼즐을 푸는 것이다.
- Unity, Arduino, Blender를 사용했다.
- ([소스 코드](https://github.com/random-entity/o.art.religious-painting-5))

Video game
{: .label }
Interactive CG
{: .label .label-green }
Sound interaction
{: .label .label-green }
Automata
{: .label .label-green }
Processing
{: .label .label-purple }
v2019-11-12
{: .label .label-yellow }

- 아이들이 운동장에 흩어지고 그룹을 짜서 놀고 있다. (현실의) 호루라기 소리가 나면 아이들은 오와 열을 맞춘 대열로 정렬한다. 호루라기를 일정 간격으로 불면, 그 리듬에 맞춰 아이들은 체조를 한다. 체조가 끝나면 아이들은 다시 흩어져 놀러 간다.
- 그래픽 디자인: 김해인; 프로그래밍: 임의존재
- Processing 및 [toxiclibs], Sound 라이브러리를 사용했다.
- [(소스 코드)](https://github.com/random-entity/o.des.horuragi)
